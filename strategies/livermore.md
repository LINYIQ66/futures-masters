# Jesse Livermore — 关键点突破策略

## 简介
- **年代**: 1877-1940
- **绰号**: 华尔街大熊、投机之王
- **传奇**: 从$5起家，1929年大崩盘中做空赚取$1亿（今日~$15亿）
- **著作**: 《股票大作手操盘术》(How to Trade in Stocks)

## 核心哲学

### 1. 关键点理论 (Pivotal Points)
Livermore 认为价格会在特定"关键点"突破，这是最佳入场时机。

**关键点类型**：
- **反转关键点**: 趋势逆转的信号点
- **持续关键点**: 趋势中的回调延续点
- **心理关键点**: 整数关口（如 $30,000）

### 2. 趋势阶段论
Livermore 将趋势分为四个阶段：

```
积累期 → 上涨期 → 分布期 → 下跌期
  (低波动)  (主趋势)  (高波动)  (反趋势)
```

**交易规则**：
- 只在"上涨期"做多
- 只在"下跌期"做空
- 在"积累期"和"分布期"观望

### 3. 最小阻力线
- 价格总是沿着"最小阻力线"运动
- 突破新高 = 多头力量强 → 做多
- 跌破新低 = 空头力量强 → 做空

## 可编程信号

### 关键点突破规则
```python
# 做多信号：突破20日最高点 + 成交量放大
if close > highest_20d and volume > avg_volume_20d * 1.5:
    signal = BUY

# 做空信号：跌破20日最低点 + 成交量放大
if close < lowest_20d and volume > avg_volume_20d * 1.5:
    signal = SELL
```

### 仓位管理
- **试探仓**: 20% 资金入场
- **确认加仓**: 价格继续朝有利方向移动后加 30%
- **趋势加仓**: 突破下一个关键点再加 50%
- **止损**: 关键点反方向 3-5%
