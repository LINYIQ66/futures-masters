1900-1940 · 华尔街早期
🎩

Jesse Livermore

1877-1940 · 趋势跟踪之父

从5美元起步，最高积累到1亿美元财富（相当于今天的15亿美元）。20世纪最伟大的投机家，被誉为"华尔街大空头"。

📈 时代背景：1900-1930年代华尔街

工业革命后的美国，铁路、钢铁、石油股票疯狂上涨。没有SEC监管，市场操纵盛行。1929年大萧条前，道琼斯指数十年上涨500%。

经典战役：1929年大萧条

在市场狂欢中，Livermore识别出顶部信号。他提前建立大量空头头寸，在1929年崩盘中单日盈利1亿美元，成为"华尔街大空头"。

1970-1980 · 滞胀时代
⚡

Larry Williams

1942- · 短线交易之王

1987年罗宾斯世界杯期货交易锦标赛冠军。用1万美元在一年内赚到110万美元，收益率达11,000%。

📈 时代背景：1970-1980年代滞胀

石油危机、高通胀、经济停滞。商品期货市场波动剧烈，为短线交易提供巨大机会。利率飙升至20%，金融市场动荡不安。

经典战役：1987年锦标赛

使用Williams %R指标识别超买超卖，结合季节性因素。在12个月内将1万美元变成110万美元，创造期货交易神话。

1970-1990 · 趋势跟踪黄金时代
🌊

Ed Seykota

1946- · 趋势跟踪传奇

12年内将5000美元增长到1500万美元，年化回报超过60%。他的学生包括多位后来的交易大师。

📈 时代背景：1970-1990年代

布雷顿森林体系崩溃后，汇率自由浮动。商品期货、外汇、股指期货蓬勃发展。计算机开始应用于交易，技术分析兴起。

经典战役：1980年代趋势跟踪

在1970-1980年代的超级趋势中，Seykota使用三重滤网系统，在白银、黄金、外汇等市场中持续盈利，创造60%的年化回报。

1980-1990 · 金融自由化
🐢

Richard Dennis

1949- · 海龟交易之父

从1600美元起步，20多岁就赚到2亿美元。发起著名的"海龟交易实验"，证明交易可以被教授。

📈 时代背景：1980年代金融自由化

里根经济学、金融市场放松管制。股指期货诞生（1982年），程序化交易兴起。大宗商品市场波动剧烈，趋势跟踪策略大放异彩。

经典战役：海龟交易实验

1983年招募23名无经验学生，教授2周海龟法则。5年内学生们赚取超过1亿美元，证明系统化交易的力量。

1987年黑色星期一

Paul Tudor Jones | 单日盈利$1亿

背景：1987年10月19日，道琼斯指数暴跌22.6%，创历史最大单日跌幅。

策略：Tudor Jones提前识别市场泡沫，通过技术分析发现市场顶部特征，建立大量空头头寸。

执行：使用期权保护多头头寸，同时建立标普500期货空头。严格止损，风险控制在2%以内。

结果：单日盈利超过1亿美元，成为黑色星期一最大赢家。

交易心理学

情绪是交易者最大的敌人，系统化执行是唯一的解药

🧠

认知偏差

  • • 确认偏差：只寻找支持自己观点的信息
  • • 锚定效应：过度依赖第一个获得的信息
  • • 幸存者偏差：只看到成功案例
  • • 近期偏差：过度重视最近发生的事件
😰

恐惧与贪婪

  • • 恐惧：亏损时过早平仓，错过反弹
  • • 贪婪：盈利时过度持仓，利润回吐
  • • FOMO：害怕错过，追高杀跌
  • • 报复交易：亏损后加大仓位试图翻本
🎯

纪律建设

  • • 制定交易计划并严格执行
  • • 设置自动止损，避免手动干预
  • • 每日交易日志，复盘总结
  • • 定期回顾，识别情绪模式
⚖️

心态管理

  • • 接受亏损是交易的一部分
  • • 关注过程而非单次结果
  • • 保持谦逊，市场永远是对的
  • • 定期休息，避免过度交易
📊

数据驱动

  • • 用统计数据替代直觉判断
  • • 回测验证策略有效性
  • • 记录每笔交易的盈亏原因
  • • 定期分析交易绩效指标
🧘

压力管理

  • • 冥想和呼吸练习
  • • 运动释放压力
  • • 保证充足睡眠
  • • 交易外培养其他兴趣

风险管理系统

活着才有机会，风控是交易的第一要务

核心风控规则

单笔风险 ≤ 2%

任何单笔交易亏损不超过总资金的2%

日度风险 ≤ 6%

单日最大亏损不超过总资金的6%

月度风险 ≤ 15%

单月最大亏损不超过总资金的15%

仓位管理公式

固定比例法

仓位 = 总资金 × 风险比例 / 止损幅度

凯利公式

f* = (bp - q) / b
b = 盈亏比, p = 胜率, q = 败率

ATR仓位法

仓位 = 风险金额 / (ATR × 乘数)

风险评估矩阵

风险等级 仓位限制 止损要求 适用场景
低风险 ≤ 5% 2% 资金止损 趋势明确，高确信度
中风险 ≤ 3% 1.5% 资金止损 震荡行情，中等确信度
高风险 ≤ 1% 1% 资金止损 不确定性高，低确信度
