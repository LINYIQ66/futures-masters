信号系统

Python实现的量化信号指标，可直接用于实盘交易

key_point_breakout.py 
def detect_breakout(data, lookback=20):
    """Livermore 关键点突破检测"""
    high = data['high'].rolling(lookback).max()
    low = data['low'].rolling(lookback).min()
    
    # 突破信号
    breakout_up = data['close'] > high.shift(1)
    breakout_down = data['close'] < low.shift(1)
    
    # 成交量确认
    vol_confirm = data['volume'] > data['volume'].rolling(20).mean()
    
    return breakout_up & vol_confirm
turtle_breakout.py 
class TurtleSystem:
    """海龟交易系统实现"""
    
    def __init__(self, entry_days=20, exit_days=10):
        self.entry = entry_days
        self.exit = exit_days
    
    def position_size(self, atr, capital, risk=0.02):
        # ATR仓位管理
        unit = capital * risk / atr
        return int(unit)
    
    def generate_signals(self, data):
        # 唐奇安通道
        upper = data['high'].rolling(self.entry).max()
        lower = data['low'].rolling(self.exit).min()
        return upper, lower

核心原则

8位大师的共同智慧，交易成功的底层逻辑

📊

趋势是你的朋友

8/8 大师都强调顺势交易。不要试图预测顶部和底部，让市场告诉你方向。

8/8 共识
🛑

止损先于入场

先想好亏多少再决定买多少。每笔交易的风险应在入场前就已确定。

风险控制第一
🚀

让利润奔跑

盈利头寸加仓，亏损头寸绝不加。截断亏损，让盈利自然增长。

盈亏比 > 胜率
🧠

情绪是最大敌人

系统化、机械化执行。恐惧和贪婪是交易者最大的敌人，用规则约束自己。

纪律 > 直觉
⚖️

仓位管理决定生死

单笔风险不超过总资金2%。活着才有机会，爆仓一次就什么都没了。

2% 规则
💰

大赚小亏

胜率不重要，盈亏比才关键。40%的胜率配合3:1的盈亏比就能稳定盈利。

3:1 盈亏比
